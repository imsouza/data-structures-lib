# Data Structures Lib

A library of data structures for academic purposes

## Usage

```
#include <stdio.h>
#include <stdlib.h>
#include "dslib.h"

int main () {
  ...
}
```

## Tests

All tests are in the [main file](https://github.com/imsouza/data-structures-lib/blob/main/tests/main.c).

#### Run tests
```
cd tests && gcc main.c ../src/*.c -o exec && ./exec
```
