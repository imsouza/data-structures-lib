Detailed Description

Huffman Coding structure definition.

The hufftree is defined as a pointer that points to a structure with the definition of two structures of the same type and two other fields of type char and int.

Warning
the fields should never be accessed directly by the program. All manipulation of data structure should be done only by the functions implemented.
