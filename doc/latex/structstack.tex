\doxysubsection{Detailed Description}
Stack structure definition. 

The stack is defined as a pointer that points to a structure with three fields and a vector of characters.

\begin{DoxyWarning}{Warning}
the fields should never be accessed directly by the program. All manipulation of data structure should be done only by the functions implemented. 
\end{DoxyWarning}
