\doxysubsection{Detailed Description}
List structure definition. 

The list is defined as a pointer that points to a structure with an entire field and the definition of a structure of the same type.

\begin{DoxyWarning}{Warning}
the fields should never be accessed directly by the program. All manipulation of data structure should be done only by the functions implemented. 
\end{DoxyWarning}
