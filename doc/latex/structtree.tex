\doxysubsection{Detailed Description}
Binary Search Tree structure definition. 

The binary search tree is defined with a character field and two other internal structures of the same type.

\begin{DoxyWarning}{Warning}
the fields should never be accessed directly by the program. All manipulation of data structure should be done only by the functions implemented. 
\end{DoxyWarning}
